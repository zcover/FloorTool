To Get Started:

Instructions:

  1. Navigate to Phone Tools, and enter your search parameters

      2. Example:
      Entering "SEA29.06" should generate a list of the employees located at SEA29 from the 6th floor.

  2. On the upper right-hand side of the page, near the search parameters, select "Export Results". A dropdown will appear, select "Export to JSON"
  3. When prompted, select the "Save File" Option, and then navigate to the downloaded file

      4. * NOTE * DO NOT RUN THIS FILE. The widget will fail if reformated by another program.

  4. Navigate back to this tool's file location (or mailroomfiles/buildings/day1/tools/FloorTool) and paste the .json file into the [logic] folder.

      5. * If a .json file already exists in logic, save over the previous fil.e

  5. To run the app, rename the .json file to "page1.json" (case sensitive)
  6. Re-launch or Refresh this web page.
